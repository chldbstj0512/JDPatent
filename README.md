# SBA2025 - 수요 기업 매칭 AI 모델

### 개요
> 사용자가 보유한 특허명세서(PDF)를 분석하여 **기술 특성 요약**, **핵심 지표 산출**,  
**관련 산업군 및 인수의향 기업 후보 추천**까지 제공하는 end-to-end AI 모델입니다.  
> 문서 분석–임베딩–매칭–추천까지 하나의 파이프라인으로 구성되어 있습니다.</br>
> 약 10만건의 특허명세서와 M&A 데이터 활용

## 1. 기술 문서 분석 (Stage 1)

사용자가 업로드한 PDF 문서를 기반으로 아래와 같은 분석 결과를 제공합니다.

- **핵심 내용 요약**  
  특허명세서 abstract -> 중요한 문장 추출

- **기술 평가 지표 산출**  
  사전 정의 factor 기반으로 특허의 강점/특성에 대한 점수를 자동 계산

- **산업 코드(NAICS 등) 매칭**  
  문서 내용 임베딩 -> 벡터DB 연결 코사인 유사도 계산 -> 가장 관련성 높은 산업 분류 코드(NAICS) 추천

---

## 2. 기업 매칭 추천 (Stage 2)

문서로부터 생성한 기술 특징 벡터를 활용하여  
해당 기술과 산업적으로 연관성이 높은 기업을 자동으로 추천합니다.

- **기술 벡터 생성**  
  자연어 기술 설명 -> 임베딩 -> 정량적 지표 변환

- **기업 데이터와 유사도 비교**  
  사전 구축 기업 벡터 데이터베이스(약 10만건)와 비교해  
  관련성 높은 기업 후보를 순위별로 추천

- **다양한 추천 전략 지원**  
  - 기술 특성 기반 추천  
  - 산업군 기반 추천
